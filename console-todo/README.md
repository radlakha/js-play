# Console Todo Manager (TypeScript)

A simple command-line task manager built step-by-step to learn JavaScript + TypeScript basics.

## How to Run

- `npm run start` → Run the app
- `npm test` → Run tests
- `npm run build` → Compile to JavaScript (outputs to `dist/`)

## Project Structure
- `src/` → Source code
- `tests/` → Test files (side-by-side with src)
- `README.md` → This file (updated after each step)

We'll build features iteratively: basic list → add tasks → priorities → persistence → etc.
